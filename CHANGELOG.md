# Changelog

Todos los cambios notables de esta plantilla se documentan en este archivo.

El formato se basa en [Keep a Changelog](https://keepachangelog.com/es-ES/1.1.0/)
y este proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [Unreleased]

## [1.1.0] - 2026-06-30

### Changed

- Fusiona las secciones **Encuéntrame** y **Contacto** en una sola sección **Contacto**
  (badges sociales + badge de email), evitando enlaces duplicados.

## [1.0.0] - 2026-06-30

### Added

- Plantilla de README de perfil con estructura moderna y corta: tagline + intro,
  Proyectos actuales, Experiencia, Stack Tecnológico, Encuéntrame y Contacto.
- Stack tecnológico en listas de texto agrupadas, con **Mobile** e **IA & Automatización**.
- Redes sociales como badges shields.io con `style=flat`.
- Guía de uso de la plantilla (`TEMPLATE-USAGE.md`) e instrucciones de uso en el README.
- Archivos de comunidad: `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `FUNDING.yml`.
- Plantillas de issues (`.github/ISSUE_TEMPLATE/`) y de pull request.
- `.editorconfig`, `.gitignore` y este `CHANGELOG.md`.
